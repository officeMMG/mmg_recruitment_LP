# 要件定義書
## 株式会社office M・M・G 採用ランディングページ

**作成日**: 2026年4月9日  
**バージョン**: 1.0

---

## 1. プロジェクト概要

### 1.1 目的
株式会社office M・M・Gの採用活動を支援するため、求職者向けランディングページを制作する。
会社の魅力・募集職種・働く環境を伝え、お問い合わせへの導線を設ける。

### 1.2 対象ユーザー
- 転職希望者
- 新卒就職活動中の学生
- 副業・パート希望者

### 1.3 成果物
- `index.html`（HTML / CSS / JavaScript をまとめた単一ファイル）

---

## 2. 機能要件

### 2.1 ページ構成

| # | セクションID | 表示名 | 概要 |
|---|---|---|---|
| 0 | - | ヒーロー | キャッチコピー・CTAボタン |
| 1 | `#about` | 私たちについて | 会社概要・経営理念 |
| 2 | `#jobs` | 募集職種 | 募集職種カード（3職種） |
| 3 | `#benefits` | 働く環境・待遇 | 福利厚生グリッド |
| 4 | `#voice` | 先輩の声 | 社員インタビューカード |
| 5 | `#contact` | お問い合わせ | 問い合わせフォーム |

### 2.2 ヘッダー

| 要件 | 詳細 |
|---|---|
| 固定表示 | `position: sticky; top: 0` でスクロール中も常時表示 |
| ロゴ | 左側に社名ロゴ（テキスト） |
| ナビゲーション | 右側にセクションリンク（5項目） |
| スクロール連動 | スクロール時にヘッダー背景色が変化 |
| レスポンシブ | モバイル時はハンバーガーメニュー表示 |
| スムーズスクロール | クリックで対象セクションへアニメーション移動 |

### 2.3 お問い合わせフォーム

#### 2.3.1 入力フィールド

| フィールド名 | 入力タイプ | 必須 | バリデーション |
|---|---|---|---|
| お名前 | `text` | ✅ | 空白不可 |
| メールアドレス | `email` | ✅ | 空白不可・メール形式チェック |
| 電話番号 | `tel` | - | なし |
| 応募職種 | `select` | ✅ | 未選択不可 |
| お問い合わせ内容 | `textarea` | ✅ | 空白不可 |
| 個人情報への同意 | `checkbox` | ✅ | チェック必須 |

#### 2.3.2 バリデーション仕様

- 送信ボタン押下時にクライアントサイドバリデーションを実行
- 未入力・未選択の必須フィールドに対して、フィールド直下に赤字エラーメッセージを表示
- メールアドレスは正規表現で形式チェックを実施
- エラーがある場合は送信処理を中断し、最初のエラー箇所へスクロール

#### 2.3.3 送信後の挙動（モック）

- 実際のメール送信・API通信は行わない
- 送信ボタン押下後、バリデーション通過時にフォームを非表示
- 「送信完了しました。担当者よりご連絡いたします。」の完了メッセージを表示

## 3. 非機能要件

### 3.1 レスポンシブデザイン

| ブレークポイント | 対応デバイス |
|---|---|
| ≥ 1024px | PC・ワイドタブレット |
| 768px〜1023px | タブレット縦 |
| < 768px | スマートフォン |

### 3.2 パフォーマンス
- 外部依存: Google Fonts（Noto Sans JP）のみ
- JavaScriptフレームワーク不使用
- 画像: CSS グラデーション・絵文字で代替（初期版）

### 3.3 アクセシビリティ
- セマンティックHTMLタグを使用（`<header>`, `<main>`, `<section>`, `<footer>` 等）
- フォームにラベル (`<label>`) を紐付け
- `alt` 属性を適切に設定

### 3.4 ブラウザ対応
- Chrome / Edge / Firefox / Safari の最新版

### 4.1 カラーパレット

| 用途 | カラーコード |

| プライマリ（紺） | `#1a3a5c` |

| テキスト（ダーク） | `#333333` |

- **本文**: Noto Sans JP (400)
- **見出し**: Noto Sans JP (700)

### 4.3 UIコンポーネント
- ボタン: 角丸・ホバーエフェクト付き
- カード: シャドウ・角丸
- セクション区切り: 交互に白・薄グレー背景

## 5. ページコンテンツ（初期ダミーテキスト）

### 5.1 ヒーロー
- キャッチコピー: 「あなたの力で、新しい価値を創ろう」
- サブコピー: 「株式会社office M・M・Gは、共に成長できる仲間を探しています」

### 5.2 募集職種（3職種）
1. 営業職（正社員）
2. 事務・バックオフィス職（正社員 / パート）
3. 企画・マーケティング職（正社員）

### 5.3 応募職種セレクトボックス選択肢
- 営業職（正社員）
- 事務・バックオフィス職（正社員）
- 事務・バックオフィス職（パート）
- 企画・マーケティング職（正社員）
- その他・一般問い合わせ
